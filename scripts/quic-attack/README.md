# QUIC Attack

## Requirements

* Linux
* Ultraman

## Run quic-attack for the default 30-second period

```bash
./scripts/quic-attack/run
```

## Run quic-attack for 5 minutes (300 seconds)

```bash
./scripts/quic-attack/run 300
```

## Run quic-attack with no timeout

```bash
./scripts/quic-attack/run 0
```
